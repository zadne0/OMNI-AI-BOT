# OMNI-AI-BOT
A multifunctional Telegram bot built with Aiogram 3.x.б using OPENROUTER and GROQ keys.
# Features
Three AI models are available, with the option to choose which one to use. A model from OpenAI called Whisper is available for listening to user voice messages. There's also a model for viewing images (using openrouterfree, as free models are quickly cut from free access). The free FLUX model for image generation is built in. An integrated SQLite database is available for storing the context of communication. Prompts for the models are also included.
# Setup
Clone this repo: git clone ... Create .env file from .env.example. Install dependencies: pip install -r requirements.txt Run: python3 main.py
